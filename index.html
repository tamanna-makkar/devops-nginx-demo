Docker Practice NGINX Demo

Hi there! This is a starter web page designed for practicing Docker build + container run workflows. It includes Dockerfile guidance and a real local health check UI section.

Features

1. Docker-ready page
Static HTML + CSS for building a simple image.
2. Command snippets
Quick Docker commands for build, run, inspect, and cleanup.
3. Service check
A sample health badge update block with dynamic status.

Docker Quickstart

# build image
docker build -t nginx-demo:latest .

# run container
docker run -d --name nginx-demo -p 8080:80 nginx-demo:latest

# test in browser
open http://localhost:8080

# logs
docker logs -f nginx-demo

# cleanup
docker stop nginx-demo && docker rm nginx-demo && docker rmi nginx-demo:latest
